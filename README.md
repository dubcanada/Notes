# Notes

A simple GTK markdown editor. Hoped to eventually be similar in features to Mou for Mac.

## Install

Requirements

* gtk+-3.0
* gtksourceview-3.0
* webkit2gtk-3.0

### Ubuntu

`sudo apt-get install libgtksourceview-3.0-dev libgtk-3-dev libwebkit2gtk-3.0-dev`

### Commands

| Type          | Command       |
| ------------- |:-------------:|
| Build and run | `make run`    |
| Build         | `mark`        |
